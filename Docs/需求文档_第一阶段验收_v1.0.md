## 1. 阶段目标

交付可运行的**竖屏微信小游戏白盒原型**，验证核心循环：

```
战斗（自动 CD） → 结算 → 商店（购买/刷新） → 下一回合战斗 → … → 第 5 回合 BOSS 通关
```
